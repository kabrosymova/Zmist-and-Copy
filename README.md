## Content Writer KPI Scorecard

Open **`content-writer-kpi-scorecard.html`** in a browser. Data is stored locally in your browser (localStorage). Use **Copy summary** or **Download PDF** from the Quality tab as needed.

To update the file from a copy one level up (optional):

```bash
cd /Users/kate/content-writer-kpi-scorecard
cp ../content-writer-kpi-scorecard.html ./content-writer-kpi-scorecard.html
```

Then commit and push as usual.
